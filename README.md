# 🚀 Releases Repository

Добро пожаловать в репозиторий релизов NULLNET Connect!
Здесь собраны все стабильные версии, сборки и сопутствующие материалы.

---

## 📦 О репозитории

Этот репозиторий предназначен для хранения:

* 📁 готовых сборок приложения
* 📝 changelog (списка изменений)
* 🏷️ тегов версий
* 📎 дополнительных ресурсов (инсталляторы, архивы и т.д.)

> ⚠️ Исходный код приложения не опубликовывается.

---

## 📥 Как скачать релиз

1. Перейдите во вкладку **Releases**
2. Выберите нужную версию
3. Скачайте подходящий файл из раздела **Assets**

Или воспользуйтесь прямой ссылкой на последний релиз:

```bash
https://github.com/nullnetvpn/nullnet-connect-platforms/releases/latest
```

---

## 🗂️ Структура релизов

Каждый релиз содержит:

* ✔️ номер версии (например: `v1.2.0`)
* 📝 описание изменений
* 📦 файлы сборки для разных платформ

Пример:

```
v1.2.0
 ├── app-win.exe
 ├── app-macos.dmg
 ├── app-linux.tar.gz
 └── checksums.txt
```

---

## 🧾 Changelog

Мы придерживаемся принципов:

* `Added` — новые функции
* `Changed` — изменения
* `Fixed` — исправления ошибок
* `Removed` — удалённый функционал

Пример:

```md
## v1.2.0
### Added
- Поддержка новой темы

### Fixed
- Исправлена ошибка авторизации
```

## 🔐 Проверка целостности

Для обеспечения безопасности:

* используйте `checksums.txt`
* проверяйте SHA256 перед установкой
